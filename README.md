# idea-matrix-app

インパクト×コストでアイデアを選別するシングルページのプロトタイプです。ブラウザだけで動作し、データはローカルストレージに保存されます。

## 使い方

1. リポジトリをローカルに取得します。
2. ルート直下の `index.html` をブラウザで開きます（ローカルファイルとして開くだけで動きます）。
3. フォームでタイトル・インパクト・コストを入力し、任意でメモを添えて保存します。
4. 保存されたアイデアは優先度スコア（`インパクト × 2 − コスト`）の高い順に並び、ゾーン判定が表示されます。
5. 「全件リセット」ボタンでローカルストレージのデータを消去できます。

## ゾーン判定ルール

- 高インパクト: 4〜5 / 低インパクト: 1〜3
- 低コスト: 1〜2 / 高コスト: 3〜5
- 高インパクト × 低コスト → 美味しい実験
- 低インパクト × 高コスト → やらない候補
- 低インパクト × 低コスト → 余裕があれば
- 高インパクト × 高コスト → 大型案件

## 開発メモ

- フレームワーク不使用のプレーン HTML + JavaScript で構成しています。
- ブラウザの LocalStorage を `ideas` キーで利用しています。
